一、方案标题
医院低成本大规模部署智能输液系统

二、方案应用场景地点：医院输液室；
功能：医院输液室经常有很多人输液，特别是在流感爆发的季节，但是医院人手又不够的情况下下。这个产品就起到了作用，它可以将输液的频率和是否缺液传送到终端，这样就可以实现一人观察整个输液是。不仅这样，它还可以实现缺液自动停止输液，远程控制停止输液。
